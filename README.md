# miniMadre

**Placa de desarrollo basada en el trabajo de Sebastian Adonay con Alejandro Rebolledo.**

## Descripción del Proyecto

El objetivo de este proyecto es desarrollar una versión miniatura y flexible de una placa madre (*motherboard*) que también funcione como una placa para el **hiriPro**.

### Características Principales

*   **Versatilidad:** Diseñada para ser una solución miniatura y flexible.
*   **Integración ESP32-S3:** Se remplaza el uso de chips FTDI por el uso directo del **ESP32-S3**, aprovechando sus capacidades nativas de USB y mayor potencia de procesamiento.
*   **Expansión Flexible:** Incorpora un conector flexible diseñado específicamente para sacar placas hijo (*daughter boards*), permitiendo una gran modularidad y adaptabilidad a diferentes proyectos.

---
*Este proyecto es una evolución colaborativa buscando optimizar espacio y funcionalidad.*
